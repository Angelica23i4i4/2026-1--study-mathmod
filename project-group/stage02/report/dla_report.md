# Теоретическое введение

Неравновесная агрегация — это процесс объединения частиц в условиях отсутствия термодинамического равновесия. Такие процессы широко распространены в природе и приводят к формированию сложных структур.

Одной из наиболее известных моделей является диффузионно-ограниченная агрегация (Diffusion-Limited Aggregation, DLA).

В данной модели:

- частицы движутся случайным образом (броуновское движение);
- при контакте с кластером частица необратимо прилипает;
- рост происходит преимущественно на границе кластера.

В результате формируются структуры, обладающие фрактальными свойствами, такими как самоподобие и дробная размерность.

# Выполнение лабораторной работы

В ходе выполнения работы была реализована модель DLA.

Алгоритм работы:

1. Инициализация начального кластера (одна точка);
2. Генерация новой частицы;
3. Случайное перемещение частицы;
4. Проверка наличия соседей в кластере;
5. Присоединение частицы;
6. Повтор процесса.

Псевдокод алгоритма:

```
C = {(0, 0)}

for i in range(N):
    p = start_position()

    while True:
        p = random_step(p)

        if is_adjacent_to_cluster(p, C):
            C.add(p)
            break
```

В процессе моделирования наблюдалось формирование разветвлённых структур.
